# react-native-menubar-extra

Add MenuBar to your App

## Installation

```sh
npm install react-native-menubar-extra
```

## Usage

```js
import { MenubarExtraView } from "react-native-menubar-extra";

// ...

<MenubarExtraView color="tomato" />
```
